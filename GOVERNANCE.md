# Governance

## Authority

This laboratory is authoritative only for its own planning history. It is not authoritative for runtime facts, architecture contracts, Direction state, release state or implementation status of `vfi64/agentic-project-kit`.

## Normative change process

1. Reviews are recorded under `reviews/`.
2. Accepted findings are documented in `DECISIONS.md`.
3. Normative specifications under `specs/dpa/` are changed only after consolidation.
4. Every repository-specific statement carries an explicit status.
5. No implementation claim is considered verified without main-repository evidence.

## Prohibitions

- No production kit code.
- No copied secrets.
- No live `.agentic/` state from private repositories.
- No fabricated test or gate evidence.
- No second runtime truth source.
